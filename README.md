# Naruto Jutsu Hand Recognition 

A simple computer vision project that triggers Naruto-style jutsu effects using hand gestures and webcam tracking.

## Features

• Real-time hand tracking
• Naruto and Sasuke jutsu effects
• Works directly in the browser
• Built using MediaPipe Hands

## Gesture Controls

Left Hand Open → Naruto Jutsu
Right Hand Open → Sasuke Jutsu

## Project Structure

```
naruto-jutsu-hand-recognition
│
├── index.html
├── README.md
└── assets
     ├── naruto.mp4
     └── sasuke.mp4
```

## Technologies Used

* JavaScript
* HTML5
* MediaPipe Hands
* Canvas API

## How to Run

1. Download or clone the repository
2. Open `index.html` in a browser or run using VS Code Live Server
3. Allow webcam access
4. Perform hand gestures to trigger jutsu effects
